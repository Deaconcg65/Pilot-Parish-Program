# Pilot Parish Initiative
**Diocese of Bridgeport · Seton Collaborative**

A dedicated program bringing expert support directly to pilot parishes — in communications, technology, data, and youth ministry.

## Site Structure

```
index.html                    ← Landing page (start here)
pilot-parish-hub.html         ← Program overview
team.html                     ← Team bios
comms/index.html              ← Communications & Design area
technology/index.html         ← Technology & Infrastructure area
images/                       ← Drop parish photos here (see README.txt)
```

## Deployment
This site deploys automatically via Netlify on push to main.

## Adding Photos
See `images/README.txt` for photo naming conventions and instructions.

## Adding New Area Pages
When Data and Youth Ministry areas are ready, create:
- `data/index.html`
- `youth/index.html`

Then update the navigation links in `index.html`, `pilot-parish-hub.html`, and `team.html`.
